Ce projet est réalisé en python avec la version 2.2.6 de JupyterLab.

Le SGBD utilisé est postgreSQL


                               Exécution du code

- Installer pgAdmin 4
- choisir 1234 comme mot de passe
- Créer une base de données nommée projet3 
 
Il suffira ensuite d'ouvrir le dossier du projet avec JupyterLab et d'exécuter le code, toutes les tables de la base de données seront automatiquement crées et remplies.
